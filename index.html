Course Description

This course will cover the basic principles of networking with a focus on protocols, implementations, and issues specific to the Internet. We will study how routing, transport, and internetworking protocols work using the Internet family of protocols as examples. We will also cover more advanced and state-of-the-art techniques such as Software-Defined Networking.

Please note that this This course will have a substantial programming component. We will learn how to read and write packets in C, gaining insight into what packets actually look like under the hood.

For further information, please read the course syllabus

Prerequisites
